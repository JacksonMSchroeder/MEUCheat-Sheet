1. OS INGREDIENTES (Variáveis e Dados)
const: Para tudo que não muda (90% do seu código).

let: Só se você precisar sobrescrever o valor (ex: um contador).

Number(dado): Transforma o texto do input em número real.

2. OS UTENSÍLIOS (Funções Arrow)
A estrutura que você vai usar para criar suas ferramentas de cálculo:

JavaScript
const nomeDaFuncao = (param1, param2) => {
    // 1. Processamento
    const resultado = param1 + param2;
    // 2. Entrega
    return resultado;
};
3. O ESTOQUE (Arrays - Listas)
Como você queria usar o [m1, m2]:

Criar: const lista = [10, 20, 30];

Acessar: lista[0] (é o 10), lista[1] (é o 20).

Tamanho: lista.length (diz quantos itens tem).

4. O MODO DE PREPARO (Estruturas de Controle)
IF / ELSE (A Trilha)
JavaScript
if (valor > 10) {
    // Faz isso
} else {
    // Faz aquilo
}
SWITCH (O Elevador)
JavaScript
switch (operacao) {
    case '+': return a + b;
    case '-': return a - b;
    default: return "Erro";
}
WHILE / FOR (A Repetição)
while: "Enquanto a condição for real, repita".

JavaScript
let contador = 0;
while (contador < 5) {
    console.log("Repetindo...");
    contador++; // IMPORTANTE: Se não somar, trava o PC!
}
5. O GARÇOM (DOM - Interação com HTML)
Como o JS "pega" e "entrega" as coisas na mesa (tela):

Pegar o valor: const x = document.getElementById('meuInput').value;

Mudar o texto: document.getElementById('titulo').innerText = "Novo Texto";

Mudar a cor: document.getElementById('caixa').style.backgroundColor = "blue";

Ouvir o clique: ```javascript
const botao = document.getElementById('btn');
botao.addEventListener('click', () => {
console.log("Alguém clicou!");
});


### 6. O INSPETOR (Testes e Debug)
* **`console.log(variavel)`**: Mostra o valor "escondido" no F12.
* **`console.table(meuArray)`**: Mostra arrays de um jeito bonitão e organizado.
* **`typeof variavel`**: Te diz se o dado é `string` (texto) ou `number`.
