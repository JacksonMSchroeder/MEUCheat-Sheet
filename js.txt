Principais Operadores Aritméticos:                                              IMPORTANTE!!!

Adição (+): Soma dois valores. Também usado para concatenar strings.

Subtração (-): Subtrai o segundo operando do primeiro.

Multiplicação (*): Multiplica dois valores.

Divisão (/): Divide o primeiro operando pelo segundo.

Módulo (%): Retorna o resto da divisão inteira entre dois números.

Exponenciação (**): Eleva o primeiro operando à potência do segundo (ES6).

Incremento (++): Aumenta o valor da variável em 1.

Decremento (--): Diminui o valor da variável em 1.

📥 1. Adicionar e Remover (Gerenciamento de Pilha)              !!IMPORTANTE para dar os comandos.. 

Perfeito para quando um novo jogador entra na lista de suspeitos ou uma carga é despachada.

.push(item): Adiciona ao final da array. (O que você mais usou até agora).

.pop(): Remove o último item.

.unshift(item): Adiciona ao início.

.shift(): Remove o primeiro item.

🔍 2. Localização e Verificação (Busca)

Ideal para saber se um Nick já está na sua lista de banidos.

.indexOf(item): Retorna a posição (índice) do item. Se não achar, retorna -1.

.includes(item): Retorna true ou false (Verdadeiro/Falso) se o item existe na lista.

.length: Não é um método, mas uma propriedade que diz o tamanho total da array.

🛠️ 3. Transformação e Filtro (O Nível "Pro")

Isso aqui vai substituir os seus for gigantes no futuro. É o que chamamos de Métodos Funcionais.

.forEach(): Executa uma função para cada item (como um for, mas mais limpo).

.filter(): Cria uma nova array apenas com os itens que passaram no seu teste (ex: denuncias > 70).

.map(): Transforma todos os itens da array (ex: pegar uma lista de preços e aplicar 10% de desconto em todos).

.find(): Retorna o primeiro item que ele encontrar que bata com a sua busca.

✂️ 4. Corte e Emenda (Manipulação)

.slice(inicio, fim): Tira uma "fatia" da array sem estragar a original.

.splice(posicao, quantosRemover): Remove ou substitui itens em qualquer lugar da array.

.join(" "): Junta todos os itens da array em uma única String (frase), separando pelo que você colocar no parêntese.

1. .map()
O que faz: Cria um novo array do mesmo tamanho, transformando cada item.
Quando usar: Quando você quer alterar os valores (ex: dobrar números, extrair nomes de objetos).


// Estrutura
const novoArray = array.map((item, index) => {
    return item * 2; // O que retornar aqui vira o novo item
});

2. .filter()
O que faz: Cria um novo array apenas com os itens que passarem no teste (True).
Quando usar: Quando você quer remover itens de uma lista (ex: pegar só números pares, ou só produtos caros).



// Estrutura
const arrayFiltrado = array.filter((item, index) => {
    return item > 10; // Se retornar TRUE, o item fica. Se FALSE, tchau.
});



3. .reduce()
O que faz: Transforma o array inteiro em um único valor (número, string, objeto ou outro array).
Quando usar: Quando você precisa de um resultado acumulado (ex: soma total, média, maior valor).



// Estrutura
const resultadoFinal = array.reduce((acumulador, atual, index) => {
    return acumulador + atual; // O que retornar vira o acumulador da próxima volta
}, 0); // <--- VALOR INICIAL (Muito importante!)

map → Transformar: Você tem 10 itens e termina com 10 itens, só que modificados.

filter → Filtrar: Você entra com 10 itens e termina com menos (ou nenhum), apenas os que passaram no seu teste.

reduce → Acumular/Condensar: Você pega a lista toda e "derrete" ela até virar uma coisa só (um total, um objeto único, uma frase).
